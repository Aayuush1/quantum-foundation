# Changelog

All notable changes to this project will be documented here.

## [1.0.0] - 2026-05-17

### Added
- 01-single-qubit-gates: X, H, Y, Z, S, T gates, Bloch sphere, custom rotations
- 02-multi-qubit-gates: CNOT, Bell states, SWAP, Toffoli, teleportation, half-adder, GHZ
- 03-quantum-algorithms: Deutsch-Jozsa, Bernstein-Vazirani, Grover's, QFT, Shor's, QPE
- 04-quantum-error-correction: Bit flip, phase flip, Shor code, threshold theorem
- 05-quantum-ml: VQE, QNN, feature maps, QAOA, quantum kernels
- 06-quantum-simulation: Harmonic oscillator, Ising model, tunneling, H2, random walks
- Unit tests for all modules
- GitHub Actions CI workflow
- Issue and PR templates
- CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md

### Notes
- All code is ASCII-only for cross-platform compatibility
- Compatible with Qiskit 1.0+
- Tested on Python 3.9-3.13
