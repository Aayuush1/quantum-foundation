# 03 - Quantum Algorithms

## What's Inside

The famous algorithms that made quantum computing known worldwide.

| Algorithm | Speedup | What It Does |
|-----------|---------|--------------|
| **Deutsch-Jozsa** | Exponential | Test if function is constant/balanced |
| **Bernstein-Vazirani** | Exponential | Find hidden string |
| **Grover's Search** | Quadratic | Search unsorted database |
| **QFT** | Exponential | Quantum Fourier Transform |
| **Shor's Algorithm** | Exponential | Factor integers (breaks RSA!) |
| **QPE** | Exponential | Estimate eigenvalues |

## Run

```bash
python quantum_algorithms.py
```

## Key Concepts

- Quantum algorithms exploit superposition and interference
- Shor's algorithm threatens modern encryption
- Grover's search is provably optimal for quantum
