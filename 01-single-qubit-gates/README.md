# 01 - Single Qubit Gates

## What's Inside

This module covers the fundamental building blocks of quantum computing.

| Demo | Description |
|------|-------------|
| **X Gate** | Quantum NOT - flips |0> to |1> |
| **H Gate** | Hadamard - creates superposition |
| **Z Gate** | Phase flip - invisible to measurement |
| **Y Gate** | Combined X+Z effect |
| **S/T Gates** | Phase rotations (pi/2 and pi/4) |
| **Bloch Sphere** | Visualize quantum states |
| **Custom Rotations** | Build any single-qubit gate |

## Run

```bash
python single_qubit_gates.py
```

## Key Concepts

- A qubit can be 0, 1, or both (superposition)
- The Bloch sphere visualizes any single-qubit state
- Any single-qubit gate = combination of X, Y, Z rotations
