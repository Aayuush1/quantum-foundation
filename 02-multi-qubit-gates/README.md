# 02 - Multi-Qubit Gates & Entanglement

## What's Inside

This module covers entanglement - the "spooky" heart of quantum computing.

| Demo | Description |
|------|-------------|
| **CNOT** | Controlled-NOT gate |
| **Bell States** | 4 maximally entangled pairs |
| **SWAP** | Exchange qubit states |
| **Toffoli** | Controlled-Controlled-NOT (AND gate) |
| **Teleportation** | Transfer states using entanglement |
| **Half-Adder** | Quantum addition circuit |
| **GHZ State** | 3-qubit entanglement |

## Run

```bash
python multi_qubit_gates.py
```

## Key Concepts

- Entangled qubits are correlated even at a distance
- Bell states are the foundation of quantum communication
- Teleportation destroys the original state (no-cloning theorem)
