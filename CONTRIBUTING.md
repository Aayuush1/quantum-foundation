# Contributing to Quantum Foundations

Thank you for your interest! This is a learning project and contributions are welcome.

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/quantum-foundation.git`
3. **Create a branch**: `git checkout -b feature-name`
4. **Make changes** and test them
5. **Commit**: `git commit -m "Add: description"`
6. **Push**: `git push origin feature-name`
7. **Open a Pull Request**

## Ideas for Contributions

- Add Jupyter notebook versions of modules
- Implement additional quantum algorithms (Simon, Quantum Counting, HHL)
- Add unit tests
- Improve visualizations
- Add more detailed explanations
- Translate comments to other languages
- Fix bugs or improve code quality

## Code Style

- Follow PEP 8
- Add docstrings to functions
- Keep ASCII-only (no Unicode characters)
- Test your code before submitting

## Questions?

Open an issue or reach out directly.
