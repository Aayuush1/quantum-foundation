# 04 - Quantum Error Correction

## What's Inside

How to protect quantum information from noise.

| Demo | Description |
|------|-------------|
| **Bit Flip Code** | 3 qubits protect against X errors |
| **Phase Flip Code** | 3 qubits protect against Z errors |
| **Shor Code** | 9 qubits protect against ANY single error |
| **Threshold Theorem** | When fault-tolerant QC is possible |

## Run

```bash
python quantum_error_correction.py
```

## Key Concepts

- No-cloning theorem makes classical error correction impossible
- Logical qubits are encoded into multiple physical qubits
- Below ~1% error rate, arbitrarily long computation is possible
