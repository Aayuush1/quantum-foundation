# 05 - Quantum Machine Learning

## What's Inside

The intersection of quantum computing and machine learning.

| Demo | Description |
|------|-------------|
| **VQE** | Find molecular ground states |
| **QNN** | Quantum neural networks |
| **Feature Maps** | Encode classical data into quantum states |
| **QAOA** | Optimization algorithm |
| **Quantum Kernels** | Similarity via quantum circuits |

## Run

```bash
python quantum_ml.py
```

## Key Concepts

- QML is still in early research stages
- VQE is a promising near-term application
- Quantum feature maps may provide advantages for certain data
