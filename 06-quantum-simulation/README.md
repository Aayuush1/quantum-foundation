# 06 - Quantum Simulation

## What's Inside

Using quantum computers to simulate quantum systems - Feynman's dream!

| Demo | Description |
|------|-------------|
| **Harmonic Oscillator** | Energy levels of quantum oscillator |
| **Ising Model** | Spin interactions and magnetism |
| **Quantum Tunneling** | Particles through barriers |
| **H2 Molecule** | Ground state energy estimation |
| **Quantum Random Walk** | Faster than classical walks |

## Run

```bash
python quantum_simulation.py
```

## Key Concepts

- Quantum computers naturally simulate quantum systems
- Classical computers struggle due to exponential state space
- H2 molecule is a promising near-term chemistry application
